Overview

A Retrieval-Augmented Generation system that combines vector search (Pinecone) with keyword search (BM25), merged via Reciprocal Rank Fusion, refined with Cohere reranking, and answered by a local LLM (Ollama) with inline citations.

Architecture

Ingestion Flow

Documents are chunked via recursive character splitting, embedded with sentence-transformers (384-dim), and upserted to Pinecone with metadata.

Query Pipeline

Vector search + BM25 run independently on the query → RRF merges ranked lists → Cohere cross-encoder reranks top candidates → Ollama generates a cited answer.

API Layer

FastAPI exposes POST /query with Pydantic validation. BM25 index rebuilt at startup; Pinecone persists in the cloud.

Key Features

  • Hybrid retrieval — combines semantic and keyword search so neither acronyms nor meaning get lost
  • Reciprocal Rank Fusion — scoreless merge using only rank positions, smoothing constant k=60
  • Cohere reranking — cross-encoder re-scores top 20 candidates with higher accuracy than bi-encoder alone
  • Local generation — Ollama runs the LLM privately, no API key needed, no data leaves your machine
  • Inline citations — answers include [Source N] markers so every claim is traceable to its source chunk
  • Deterministic chunk IDs — MD5-hashed IDs prevent duplicate upserts on re-ingestion
